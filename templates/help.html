Help

Here's some information about what this Poll Manager is about, as well as the embeddable widget that is created for each poll


Poll Manager

Features

  1. View all open and closed polls
  2. Create a new poll
    • 2 compulsory answers to be added to the poll
    • 2 optional answers can be added to the poll
  3. View the results for a poll
Widget

Features

  1. User can cast their vote
  2. User can view the current percentages for each answer - once they've voted
